---
title: Installation
description: Install Minions Qa-rules packages
---

## TypeScript / Node.js

```bash
npm install @minions-qa-rules/sdk minions-sdk
```

## Python

```bash
pip install minions-qa-rules
```

## CLI

```bash
npm install -g @minions-qa-rules/cli
```

## Requirements

- **Node.js** 20 or later
- **Python** 3.11 or later (for Python SDK)
