# Minions Qa-rules

Validation rules, consistency checks, compliance gates, and auto-fix definitions

## Overview

This project is part of the [Minions ecosystem](https://github.com/mxn2020/minions) and provides:

- **TypeScript SDK** (`@minions-qa-rules/sdk`) — Core library for Node.js
- **Python SDK** (`minions-qa-rules`) — Python bindings
- **CLI Tool** (`@minions-qa-rules/cli`) — Command-line interface
- **Web App** — Interactive playground at [qa-rules.minions.wtf](https://qa-rules.minions.wtf)
